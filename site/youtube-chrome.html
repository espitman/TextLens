TextLens YouTube Chrome

Your own subtitles, inside YouTube.زیرنویس خودت، داخل یوتیوب.

A lightweight Chrome extension for loading SRT files, styling subtitles, and keeping translated text synced with the video player.

یک اکستنشن سبک کروم برای لود فایل SRT، تنظیم ظاهر زیرنویس و هماهنگ نگه داشتن متن ترجمه با پخش ویدیو.

Download extensionBack to hub
Custom subtitles render directly on the video.

Why Chrome

The browser is where many users already watch and learn. The extension keeps the experience close to the video instead of moving it into another tool.

مرورگر جایی است که بسیاری از کاربران تماشا و یادگیری را انجام می‌دهند. اکستنشن تجربه را کنار ویدیو نگه می‌دارد، نه در یک ابزار جدا.

Distribution

Packaged as a ready-to-load extension zip; no npm install and no Chrome Web Store dependency for the experimental release.

به صورت zip آماده لود عرضه شده؛ برای ریلیز آزمایشی نه npm install لازم دارد و نه وابسته به Chrome Web Store است.

Launch state

Available as a GitHub release package for manual Chrome loading.

به صورت پکیج ریلیز گیت‌هاب برای لود دستی در کروم آماده است.
