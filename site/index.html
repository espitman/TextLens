Product suite

این مجموعه خواندن شخصی، ترجمه صفحه موبایل، تماشای یوتیوب، آماده‌سازی زیرنویس، تولید زیرنویس روی اندروید و پخش روی تلویزیون را پوشش می‌دهد.

M TextLens macOS

Instant screen translation for Mac

A menu bar app for selecting any screen area and reading Persian translation in a polished popup.

 A TextLens Android

Floating OCR for Android apps

A movable bubble that lets users translate visible text without leaving the current app.

یک بابل شناور که بدون خروج از اپ فعلی، متن روی صفحه را ترجمه می‌کند.

 TV TextLens TV

Subtitle control for Android TV

A TV overlay and web control panel for rendering translated SRT files over YouTube playback.

اورلی تلویزیون و پنل وب برای نمایش زیرنویس ترجمه‌شده روی پخش یوتیوب.

 YT TextLens YouTube Chrome

Custom subtitles inside YouTube

A Chrome extension for loading SRT files and rendering them directly on YouTube pages.

اکستنشن کروم برای لود فایل SRT و نمایش مستقیم آن روی صفحات یوتیوب.
