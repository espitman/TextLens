TextLens

Select a piece of your screen, let TextLens read it with OCR, and get a Persian translation in a fast, draggable popup built for day-to-day reading.

Download DMG How it works

Selected article text

TextLens sits quietly in the menu bar. Press the shortcut, drag over any readable area, and keep working while OCR and translation run.

It is designed for screenshots, web pages, PDFs, presentations, and any app where text cannot be copied directly.

Provider settings are available from the popover with separate OpenRouter and Liara profiles.

Area selection

Use the global shortcut to draw over exactly the text you want, with a custom TextLens cursor.

Persian-first popup

Translations open instantly in a compact RTL popup with copy, retry, cost badge, and drag support.

Provider control

Switch between OpenRouter and Liara, choose the model, and retry failed translations without reopening settings.

Install on macOS

Download the latest DMG, drag TextLens into Applications, grant Screen Recording permission, then use Command + Shift + 0 from anywhere.

  1. Open the DMG and move TextLens to Applications.
  2. Open TextLens from the menu bar and configure an API provider.
  3. Press Command + Shift + 0, select text on screen, and translate.
TextLens is an early macOS release. Android and more platforms are planned later.
